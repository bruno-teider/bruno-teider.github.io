Search for The Sword

Esse projeto é um jogo feito na plataforma Construct 3, nele é contado a história do guerreiro Anaril, que perde tudo por causa do ataque de um dragão em sua vila. O jogo apresenta um mix de plataforma com runner, contendo 5 fases e diversas cutscenes que contam uma empolgante história.

Site de Receitas

Um site de receitas contendo 4 receitas de bolo e uma página onde o usuário pode enviar sua própria receita e ver as receitas enviadas. Este é um site básico, entretanto ele contém uma ligação com um banco de dados MySQL. O projeto foi feito utilizando HTML, CSS, PHP e MySQL.

Only The Skates Know

Um dos primeiros projetos em que trabalhei. O jogo Only The Skates Know (OTSK, abreviado) foi feito pela plataforma Scratch e apresentou duas versões. Eu e minha equipe não ficamos muito satisfeitos com o resultado final da versão original e decidimos fazer um remake do jogo, onde polimos as mecânicas e programação, alteramos todas as artes e cutscenes e mudamos boa parte da história. Um exemplo de mudança que fizemos é que na versão original o personagem principal se transformava em um arqueiro, enquanto no remake ele ganha uma espada. Infelizmente, a versão original foi perdida.(Diferente do Search for The Sword, aqui praticamente todas as artes foram feitas a mão pela equipe).

Loja Virtual OTSK

Essa é a loja virtual de produtos do jogo Only The Skates Know. A loja apresenta algumas variedades de canecas e camisetas e páginas sobre os fornecedores. Esse é um projeto bem simples, porém foi o primeiro site que fiz contendo ligação com banco de dados. Ele foi feito utilizando HTML, CSS, JavaScript e ligação com banco de dados MySQL.

Plantae

Sem dúvidas o Plantae é o maior projeto que já fiz/participei, ele é uma estufa automatizada para o crescimento de plantas indoor, contendo iluminação automática, sensores de umidade no solo para fazer irrigação, sensor de temperatura e um cooler, para resfriar a planta, aviso sobre nível de água no reservatório e uma conexão com um site onde o usuário pode ver os dados em tempo real. O projeto foi feito utilizando Arduino e outros componentes para o modelo físico, o corpo do Plantae foi feito usando corte a laser.

Site do Plantae

O site do Plantae conta com uma landing page que explica um pouco sobre o projeto, uma página de login, cadastro e uma página onde o usuário pode ver os status da sua planta. Cada produto teria um código, usando ele o usuário poderia se cadastrar e adicionar um novo dispositivo, assim podendo ter vários Plantaes em casa.
